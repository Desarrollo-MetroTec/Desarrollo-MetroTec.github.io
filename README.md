# Generador de Firmas de Correo - MetroTec

¡Bienvenido al **Generador de Firmas de Correo** oficial de MetroTec! 

Esta es una herramienta sencilla y rápida diseñada para que todos los colaboradores de la empresa puedan crear su propia firma de correo electrónico institucional de manera estandarizada y profesional.

## ¿Qué hace esta herramienta?

El generador te permite:
- **Llenar tus datos personales:** Nombre, cargo, teléfono y correo electrónico.
- **Añadir tu foto de perfil:** Puedes subirla usando un enlace (incluso directamente desde Google Drive).
- **Ver cómo queda en vivo:** A medida que escribes, puedes ver una vista previa de cómo se verá tu firma terminada.
- **Obtener tu firma lista para usar:** Con un solo clic, se genera el formato perfecto para que lo copies y pegues directamente en la configuración de firmas de tu cliente de correo (como Gmail o Outlook).

## ¿Cómo usarlo?

1. Abre el archivo principal en tu navegador web.
2. Completa los campos con tu información. (Nota: Si dejas el campo de teléfono vacío, simplemente no aparecerá en la firma).
3. Haz clic en **"Generar Firma"**.
4. Haz clic en **"Abrir Código"**. Se abrirá una nueva pestaña con tu firma terminada.
5. Selecciona todo en esa nueva pestaña, cópialo y pégalo en el editor de firmas de tu correo electrónico.
